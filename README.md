

# BAALBEK-GEO

## Questioning the Original Intention of the Baalbek Quarry Megaliths

**Author:** Alan G. Friar
**Organization:** EnBra Group
**Version:** 1.0
**Date:** 2026-04

---

This repository contains a technical discussion paper examining the original intention of the Baalbek quarry megaliths, with particular focus on Hajar al-Hibla.

The analysis presents independently verifiable geodetic measurements, orientation data, and archaeoastronomical observations to evaluate whether the stones may have served a fixed reference function rather than purely a construction role.

Rather than proposing a definitive conclusion, the paper establishes a structured framework of measurable observations, identified patterns, and testable hypotheses intended to support further empirical investigation.

## Structure of Analysis

The paper is organized into the following sections:

1. **Context and Central Question**
   Site background and definition of the research problem

2. **Physical Characteristics**
   Quarry stone properties, orientation, and anomalies

3. **Astronomical Considerations**
   Precessional alignment and stellar correlations

4. **Geodetic Observations**
   Distance relationships and global alignment patterns

5. **Comparative Linguistics**
   Cross-cultural terminology and symbolic parallels

6. **Instrumentation Question**
   Precision requirements and missing measurement systems

7. **Mainstream Interpretation Review**
   Comparison with existing archaeological explanations

8. **Proposed Investigations**
   Ten testable research pathways

9. **Collaborative Framework**
   Relevant researchers and institutions

## Key Characteristics

* All distances calculated using the haversine formula (R = 6,371 km)
* All bearings calculated using standard great-circle methods
* All values independently reproducible
* Clear separation between:

  * measured data
  * observed patterns
  * hypotheses
